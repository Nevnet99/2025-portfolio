# Modern CSS Techniques for Better Web Design

Exploring the latest CSS features and techniques that are transforming web design. From Grid to Custom Properties, let's dive into modern CSS.

## CSS Grid Layout

CSS Grid has revolutionized web layouts:

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
}
```

## Custom Properties (CSS Variables)

Dynamic theming with CSS variables:

```css
:root {
  --primary-color: #0066cc;
  --secondary-color: #ff4400;
}

.button {
  background: var(--primary-color);
  color: white;
}

.button:hover {
  background: var(--secondary-color);
}
```

## Container Queries

The future of responsive design:

```css
@container (min-width: 400px) {
  .card {
    grid-template-columns: 2fr 1fr;
  }
}
```

## Modern Selectors

Using powerful new selectors:

```css
/* Style every other card */
.card:nth-child(even) {
  background: #f5f5f5;
}

/* Style based on data attributes */
[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #ffffff;
}
```

## Conclusion

Modern CSS provides powerful tools for creating flexible, maintainable, and beautiful web designs. Stay updated with these techniques to build better user experiences.
